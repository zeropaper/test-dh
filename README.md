# test-dh

A test job for a company I'd like to work for.

## Install

You will need git, node.js and npm installed and then

````sh
git clone https://github.com/zeropaper/test-dh.git
cd test-dh
npm i
````

## Build

First, install (see above) and then:  
`npm run build`

## Development

First, install (see above) and then:  
`npm run start`
